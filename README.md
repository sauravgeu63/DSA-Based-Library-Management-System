# DSA-Based-Library-Management-System
**DSA-Based Library Management System** A console-based project built using **C and Python** that manages books and users efficiently. It applies **linked lists** for dynamic book records, **queues** for issue/return requests, and **stacks** for tracking recent activities, showcasing core **DSA concepts** in a practical system.



** Project Execution Steps **
1. In main.py file open a new or an existing terminal and type " gcc -shared -o library.dll library.c "
2. After that click enter and type python main.py and enter .
3. A project dashboard will be opened in your desktop.

4. Ensure all files should be in a single folder also.
